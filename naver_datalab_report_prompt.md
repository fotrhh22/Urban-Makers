# GPT용 창신숭인 도시재생 온라인 반응 보고서 작성 프롬프트 (V4_No_Sentiment)

이 프롬프트는 긍/부정 감성 지표를 완전히 제거하고 골목경관 지표를 신설하여 보정한 데이터를 바탕으로, **창신·숭인동 도시재생 온라인 반응 관심도 조사 보고서**를 완벽하게 작성하도록 유도하는 전문가용 프롬프트입니다. GPT 등 LLM 모델에 그대로 입력하여 사용하실 수 있습니다.

---

```markdown
당신은 도시공학, 도시행정, 그리고 빅데이터 분석 전문가입니다. 아래 제공된 네이버 검색 트렌드 API 분석 데이터를 바탕으로 **"창신·숭인동 도시재생 선도사업에 대한 연도별 온라인 반응 및 관심도 조사 보고서"**를 논리적이고 학술적인 정책 보고서 톤앤매너로 작성해 주세요.

## 1. 분석 배경 및 목적
- 창신숭인 지역은 2014년 전국 1호 도시재생 선도지역으로 지정되어 2018년까지 마중물 사업이 집중 추진되었고, 이후 관리 및 자립 단계에 접어들었습니다.
- 본 분석은 네이버 검색 API(데이터랩)를 활용하여 2012년부터 2025년까지 창신숭인 지역 및 세부 도시재생 요소들에 대한 대중의 검색 트렌드 변화를 분석함으로써, 도시재생 사업의 실질적 전파력과 대중적 관심도를 측정하고자 합니다.
- **데이터 한계:** 네이버 데이터랩 API 정책상 2016-01-01 이전 데이터는 수집이 불가하여 2012~2015년은 '수집 불가(null)' 처리되었으며, 실질 분석은 2016~2025년(10개년)을 대상으로 합니다.

## 2. 최종 Proxy 해석 정의 (필수 반영)
최종 도시재생 온라인 반응 Proxy는 거점공간, 방문관광, 골목경관, 봉제산업의 네 구성요소만 포함합니다. 긍정반응과 부정불편은 각각 방문관광 및 생활환경 키워드와 의미상 중복되고 값의 변동 폭이 작아 최종 Proxy에서 제외하였습니다. 따라서 본 Proxy는 감성분석 지표가 아니라 창신숭인 도시재생 콘텐츠가 온라인상에서 얼마나 장소·방문·경관·산업 대상으로 소비되었는지를 보는 검색 기반 관심 지표로 해석해야 합니다.

## 4. 분석 방법론 및 보정 원리
1. **다중 배치 보정 (Calibration):** API의 그룹 개수 제한(최대 5개) 극복을 위해 Core/Context 배치로 나누어 수집했습니다. 공통 그룹 '창신숭인_지역인지도'를 기준으로 스케일 보정 계수($F = 1.0000$)를 구해 통합 스케일로 보정했습니다.
2. **지표 제외 설계:** 단순 행정구역 검색어(길 찾기 등)인 `area_search_attention_index`를 Proxy에서 제외하여, 도시재생의 실제 콘텐츠 효과만을 왜곡 없이 포착했습니다.
3. **신규 가중합 공식:** 
   $$\text{Urban Regen Proxy} = \text{거점공간}(0.35) + \text{방문관광}(0.30) + \text{골목경관}(0.25) + \text{봉제산업}(0.10)$$
4. **신뢰도 검증 (Coverage):** 연도별 데이터 수집율이 75% 미만인 구간(2023년 및 2025년 봉제산업 지표 일부 월 누락)에는 `low_coverage_warning`을 부착하여 분석에 신중을 기했습니다.
```
